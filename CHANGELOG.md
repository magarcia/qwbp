# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-01-16

### Added
- Initial beta release
- QWBPConnection class for high-level connection management
- QWBPEncoder/QWBPDecoder for binary packet encoding/decoding
- SDP reconstruction utilities
- HKDF-based ICE credential derivation
- Support for IPv4, IPv6, mDNS, and TCP candidates
